# TD3

## Exercice 1

1. 
'bab + aba' est accepté par l'automate.

2. 
'bab' est un mot accepté, 'baa' n'est pas accepté

3. 
w = 'aabaabbabbabaaba'

'aabaab' est le plus long préfixe accepté.
il reste donc 'babbabaaba'

'babbaba' est le plus long préfixe accepté du mot restant.
il reste donc 'aba'

'aba' est accepté.
